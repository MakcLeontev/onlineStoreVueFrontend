# интернет магазин
Бекенд реализован на Spring Boot 
https://github.com/MakcLeontev/onlinestore.git

## Установить 
```
npm install
```

### Запустить
```
npm run serve
```
### Проект будет доступен по адресу 
http://localhost:8081/

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
